Blocked Keyword Finder (Only tested with goguardian)

1. Navigate to chrome://history or press ctrl+h.

2. Enter in any keyword into the history search bar.

3. If the history page turns into a blocked page, then that keyword is blocked.

Note: You want to use chrome://history instead of searching a keyword on google because they can't see what you search in chrome://history.

Disable tab limit

Note 1: Only tested for goguardian.

Note 2: You need to do the steps again if goguardian disconnects then reconnects.

Note 3: This works with any website that when trying to close says "are you sure you want to leave this site?"

Note 3.1: Websites that show the message: Iready, The home page for this site

Note 4: Skip steps 1 and 2 if using a different website than Iready.

Note 4.1: If using Iready fothis and the Iready tab shows the alert, it still works.

1. Open Iready

2. Get onto a mypath lesson

3. You need to make sure you have as many tabs open as it will allow you to before doing this next step

4. Make sure the Iready (or other website) tab is at the end of all of your tabs

5. Right-click on the tab before Iready (or other website) and press "new tab to the right"

6. If you did all the steps correctly, it should bring you to the Iready tab (or other website) and ask you if you want to leave the site

7. Press cancel when it asks you

8. You are now able to open infinite tabs! You can even close the Iready (or other website) tab too.
